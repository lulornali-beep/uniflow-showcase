## 🚀 快速开始

### 1. 启动管理后台

```bash
cd admin-console
npm install
npm run dev
```

访问：`http://localhost:3000`

### 2. 登录系统

使用 Supabase 配置的管理员账号登录。

### 3. 使用 AI 采集台

参考 [AI 采集台使用说明](./AI_采集台使用说明.md)
