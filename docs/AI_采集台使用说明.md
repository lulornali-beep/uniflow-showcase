## 🔧 配置要求

### 环境变量

确保 `.env.local` 文件中包含 DeepSeek API Key：

```env
DEEPSEEK_API_KEY=your_deepseek_api_key_here
```

**获取 API Key**：
1. 访问 https://platform.deepseek.com/
2. 注册/登录账号
3. 进入 API Keys 页面
4. 创建新的 API Key
5. 复制到 `.env.local` 文件

---

## 🚀 使用方法

### 步骤 1: 输入内容

1. 选择输入类型：
   - **文本**：粘贴微信聊天记录或其他文本内容
   - **链接**：粘贴公众号链接或网页链接
   - **图片**：暂不支持（功能待完善）

2. 在输入框中粘贴内容

### 步骤 2: AI 识别

1. 点击 "AI 识别" 按钮
2. 等待 AI 处理（通常需要 5-15 秒）
3. 查看 AI 处理日志

### 步骤 3: 人工校对

1. 在右侧 "AI 识别结果" 区域查看提取的信息
2. 修改需要调整的字段：
   - 标题
   - 类型（招聘/活动/讲座）
   - 关键信息（公司、岗位、日期、时间、地点、截止时间、链接等）
   - 摘要
   - 标签

### 步骤 4: 发布或保存

1. **保存草稿**：将内容保存为草稿状态
2. **确认发布**：直接发布到小程序（功能待完善）

## 📋 提取的信息字段

AI 会自动提取以下信息：

- **标题**：活动或招聘标题（包含公司名称和岗位名称）
- **类型**：recruit（招聘）/ activity（活动）/ lecture（讲座）
- **关键信息**：
  - 公司名称
  - 岗位名称
  - 日期
  - 时间
  - 地点
  - 截止时间
  - 投递链接
  - 是否内推
- **摘要**：一句话摘要（50字以内）
- **标签**：3-5 个相关标签

## 🐛 常见问题

### 问题 1: AI 识别失败

**可能原因**：
- DeepSeek API Key 未配置或错误
- 网络连接问题
- 内容格式不符合要求

**解决方案**：
1. 检查 `.env.local` 文件中的 `DEEPSEEK_API_KEY` 是否正确
2. 检查网络连接
3. 重启开发服务器（`npm run dev`）
4. 查看浏览器控制台的错误信息

### 问题 2: URL 抓取失败

**可能原因**：
- 链接需要验证（微信公众号）
- 链接已失效
- 网络超时

**解决方案**：
1. 对于需要验证的微信公众号链接，建议手动复制文章内容作为文本输入
2. 检查链接是否有效
3. 重试几次

### 问题 3: AI 提取的信息不准确

**说明**：这是正常的，AI 识别可能不够准确。

**解决方案**：
1. 使用右侧编辑功能手动修正
2. 在原始素材中补充更详细的信息
3. 多次尝试不同的表达方式
