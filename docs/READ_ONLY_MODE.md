# 📖 只读模式说明

## ✅ 已实现的只读展示

现在公开展示版本是**只读模式**，可以查看所有功能界面，但不能修改数据。

## 🌐 访问地址

```
http://localhost:3000
```

## 📋 功能对照表

### ✅ 可以使用的功能

| 功能 | 说明 | 可用性 |
|------|------|--------|
| **活动管理** | 查看所有活动列表 | ✅ 完全可见 |
| 筛选功能 | 按状态、类型筛选 | ✅ 可用 |
| 搜索功能 | 搜索标题、摘要、标签 | ✅ 可用 |
| 查看详情 | 点击查看活动完整信息 | ✅ 可用 |
| **数据看板** | 查看统计数据 | ✅ 完全可见 |
| 统计卡片 | 总数、已发布、草稿、已下架 | ✅ 可见 |
| **AI 采集台** | 查看功能说明 | ✅ 可见 |

### ❌ 不可用的功能（已隐藏或禁用）

| 功能 | 原因 | 状态 |
|------|------|------|
| 新建活动 | 会修改数据 | ❌ 已隐藏 |
| 编辑活动 | 会修改数据 | ❌ 已隐藏 |
| 删除活动 | 会修改数据 | ❌ 已隐藏 |
| 置顶功能 | 会修改数据 | ❌ 已隐藏 |
| 状态切换 | 会修改数据 | ❌ 已隐藏 |
| AI 解析 | 会创建新数据 | ❌ 仅展示说明 |

## 🔒 数据安全

### 只读模式的保护
- ✅ **不会修改数据库**：所有修改操作都已移除
- ✅ **只能查看**：只有查询功能可用
- ✅ **安全展示**：可以放心给任何人看

### 与管理后台的区别
| 项目 | 只读模式 | 管理后台 |
|------|---------|----------|
| 访问地址 | `/showcase/*` | `/dashboard/*`, `/events/*` |
| 登录要求 | 无需登录 | 需要登录 |
| 查看数据 | ✅ | ✅ |
| 修改数据 | ❌ | ✅ |
| 删除数据 | ❌ | ✅ |
| AI 解析 | ❌ | ✅ |

## ❓ 常见问题

### Q: 为什么 AI 采集台不能用？
A: 因为 AI 解析会创建新的活动数据，只读模式不允许修改数据。

### Q: 可以看到所有活动吗？
A: 可以，包括已发布、草稿和已下架的活动都能看到。

### Q: 数据是实时的吗？
A: 是的，显示的是真实数据库中的数据。

### Q: 如何切换到完整功能？
A: 访问 `/login` 登录管理后台即可使用完整功能。

### Q: 只读模式安全吗？
A: 非常安全，所有修改操作都已移除，不会影响数据库。
